# Install
You can compile the code yourself, which is better, or use the .jar file in the release tab. Use the largest .jar file, which ends with "beta" if you compile it. And then put the .jar file from the build folder after compilation into your mod folder of your Forge 1.12.2 instance. And for when you're in the game, press the "y" key to open the mistletoe.
